OC Branch Line ( Also named OpenCore Branch Line ) is the first subway line in GNU/Genshin subway world. It begins from north terminal, the OC First House, and terminates at south terminal, the South Power Plant Station. Currently, it's still build in progress (BIP). The length of this line will be 98 KM long when the construction finished.
